# Agent Guidelines

## Rules

1. **Conventional Commits** — every commit must use `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, or `style:` prefix. Imperative mood, lowercase, under 72 chars.
2. **Update README on every feature change** — if you add, change, or remove a feature, update `README.md` to reflect it. No exceptions.
